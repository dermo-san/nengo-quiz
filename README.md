# 年号120

小学生向けの歴史年号暗記用静的PWAです。vanilla HTML/CSS/JSのみで動き、ビルドや外部リソースは使いません。

## 起動方法

`file://` で開くとService Workerが動かないため、ローカル配信で確認します。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開きます。

## 問題データ

`questions.sample.json` を初期データとして同梱しており、起動時に自動で読み込みます（現在は「重要年代（標準編）120問」の本データ）。別のデータに差し替えたいときは、アプリの「設定とデータ」画面からJSONファイルを読み込みます。「初期データに戻す」で同梱データを再読み込みできます。

問題データの形:

```json
{
  "version": "2026-07-08",
  "title": "重要年代（標準編）120問",
  "questions": [
    { "id": 1, "col": 1, "row": 1, "round": 1, "event": "出来事", "year": 645, "goro": "" }
  ]
}
```

読み込み時に、120問・id 1〜120・year整数・round 1〜4・列と回の対応を検証します。不正なJSONは反映されません。

## 練習モード

ホームで出題数を `30問` / `15問` から選べます。

- 30問: 回別は2列30問、苦手優先と全範囲ランダムも30問です。
- 15問: 回別は前半/後半のどちらか1列15問、苦手優先と全範囲ランダムも15問です。

問題は初回3連続正解でマスターになります。マスター後に間違えると解除され、再マスターは2連続正解で復帰します。

## 記録の保存

問題データ、問題別成績、セッション履歴、設定はブラウザのlocalStorageに保存します。「設定とデータ」画面からJSONとしてエクスポート/インポートできます。

## GitHub Pages公開手順

1. このディレクトリをGitHubリポジトリにpushします。
2. GitHubのSettings → Pagesで、公開ブランチとルートディレクトリを選びます。
3. 公開URLをiPad Safariで開き、ホーム画面に追加します。
4. 問題データ（`questions.sample.json`）は同梱・公開されるため、ホーム画面追加後はそのまま使えます（ファイル読み込みは不要）。
